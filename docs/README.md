# 派克ERP

#### 介绍
派克ERP采用前后端分离式架构开发，致力于为商家提供高效专注的电商ERP系统。


#### 特性
- 简洁优雅的交互界面
- 高效易用的功能设计
- 专注于电商的轻量化
